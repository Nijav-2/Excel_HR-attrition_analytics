# 📊 HR Attrition Analysis Project

## 📌 Overview  
This project focuses on analyzing **employee attrition patterns** using Microsoft Excel. The goal is to understand how factors such as age, experience, and job roles influence employee turnover, helping organizations improve retention strategies.

All reports and dashboards created for this project are **exported and presented in PDF format** for easy access and sharing.

---

## 🎯 Objectives  
- Analyze employee attrition trends  
- Understand the relationship between age and experience  
- Identify job roles with higher attrition rates  
- Support HR decision-making for employee retention  

---

## 🔍 Key Analysis Performed  

### 👨‍💼 Employee Demographics Analysis  
- Experience in current role categorized by age  
- Analysis of workforce distribution across age groups  

### 📉 Attrition Analysis  
- HR attrition rate by age  
- Identification of age groups with higher employee turnover  

### 🧑‍💻 Job Role Analysis  
- Analysis of attrition across different job roles  
- Identification of roles with higher attrition rates  

---
